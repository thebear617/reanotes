所有"编码器→中间层→解码器"结构的模型可以划分为两大流派：

|  | 自编码器（AE） | 密集预测架构（U-Net） |
| --- | --- | --- |
| 信息通道 | 只能走瓶颈（bottleneck） | 瓶颈 + 跳接连路（skip connection） |
| 核心机制 | 信息压缩 → 迫使学到本质表示 | 上下文+定位 → 像素级对齐 |
| 学习方式 | 自监督（重构自身） | 监督（逐像素预测） |
| 学什么 | 语义表示 / 紧凑编码 | 空间对应关系 |
| 最怕什么 | 网络"作弊"不走瓶颈 | 空间细节在下采样中丢失 |
| 代表 | AE, VAE, MAE, DAE | U-Net, U-Net++, Attention U-Net |
