title: 完整的归属链
tag: [分类]
---

**自编码器**

- 架构上 → 编码器-解码器架构
- 任务上 → 表示学习 / 重构范式
- 设计上 → 信息压缩范式（Compression via Bottleneck）
- 学习上 → 自监督学习

**U-Net**

- 架构上 → 编码器-解码器架构
- 任务上 → 密集预测范式（Dense Prediction）
- 设计上 → 上下文+定位范式（Context + Localization via Skip Connections）
- 学习上 → 监督学习

它们共享一个父亲（编码器-解码器），但母亲（任务目标）完全不同——就像螺丝刀和开瓶器，都是"手持工具"，但设计语言截然不同。
