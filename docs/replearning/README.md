表征学习研究模型如何把原始数据映射为可用于分类、生成、检索或决策的表示。这里不再把“监督、自监督、对比、生成”放在同一层级，而是用两个相互独立的问题定位方法：

- **表征约束**：希望表示保留什么信息、形成什么结构？
- **信号来源**：训练目标由谁、由什么过程产生？

具体的损失函数、网络结构和训练技巧作为第三层“实现机制”，不与前两轴混为一谈。

## 总体结构

```text
一个表征学习方法
├── 主轴：表征约束
│   ├── 重建—生成
│   ├── 预测—上下文
│   └── 对齐—关系
├── 副轴：训练目标的构造来源
│   ├── 人工标注
│   ├── 天然配对
│   ├── 数据自指
│   ├── 模型自举
│   └── 环境反馈
└── 第三层：实现机制
    ├── 对比学习
    ├── 自蒸馏
    ├── 冗余消除
    ├── 掩码建模
    └── 生成建模等
```

主轴允许多标签；副轴记录一个主要来源和必要的辅助来源。同一个方法可以跨越多个目标，也可能组合多种训练信号。
