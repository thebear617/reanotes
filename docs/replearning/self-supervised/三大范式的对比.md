---
title: 三大范式的对比
tag: [对照]
---

|          | 生成式              | 对比式             | 非对比式                 |
| -------- | ------------------- | ------------------ | ------------------------ |
| 代表     | AE, MAE, BERT       | SimCLR, MoCo, CLIP | BYOL, Barlow Twins, DINO |
| 监督信号 | 重构/预测被破坏部分 | 正负样本对的区分   | 正样本对的相关性约束     |
| 负样本   | 不需要              | 需要               | 不需要                   |
| 学到什么 | 数据的分布 / 结构   | 判别性边界         | 去相关的语义空间         |
| 坍塌风险 | 低（有重构目标）    | 低（有负样本推开） | 中（靠正则化防坍塌）     |
