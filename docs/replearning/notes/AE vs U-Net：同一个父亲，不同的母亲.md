---
title: AE vs U-Net：同一个父亲，不同的母亲
tag: [架构, 对照]
---

用家族比喻来记住它们的关系：

| 对照维度 | 自编码器           | U-Net              |
| -------- | ------------------ | ------------------ |
| 父亲     | 编码器-解码器架构  | 编码器-解码器架构  |
| 母亲     | 表示学习（自监督） | 密集预测（监督）   |
| 天赋     | 学通用特征，可迁移 | 像素级对齐，精度高 |
| 弱点     | 瓶颈可能丢失细节   | 学到的表示难迁移   |
| 比喻     | 螺丝刀（干啥都行） | 开瓶器（开瓶一绝） |
