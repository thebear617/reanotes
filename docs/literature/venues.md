## 顶级会议

| 名称                                      | 方向        | 说明                                                         |
| ----------------------------------------- | ----------- | ------------------------------------------------------------ |
| [NeurIPS](https://neurips.cc/)            | ML · 多模态 | 表征学习、自监督学习论文的重镇；多模态表征学习常见于此。     |
| [ICML](https://icml.cc/)                  | ML          | 机器学习综合顶会，表征学习论文数量大。                       |
| [ICLR](https://iclr.cc/)                  | 表征学习    | 名称直接包含 Representations，表征学习论文最集中的会议之一。 |
| [CVPR](https://cvpr.thecvf.com/)          | CV          | 表征学习聚焦视觉领域的主场，包括对比学习和自监督视觉预训练。 |
| [ICCV](https://iccv.thecvf.com/)          | CV          | 视觉领域顶会，自监督视觉预训练论文集中。                     |
| [ECCV](https://eccv.ecva.net/)            | CV          | 视觉领域顶会。                                               |
| [ACL](https://www.aclweb.org/)            | NLP         | NLP 表征学习，包括词向量、句子表征和预训练语言模型。         |
| [EMNLP](https://2026.emnlp.org/)          | NLP         | NLP 实证方法顶会，预训练语言模型论文密集。                   |
| [NAACL](https://naacl.org/)               | NLP         | ACL 北美分会，NLP 表征学习论文集中。                         |
| [AAAI](https://aaai.org/conference/aaai/) | AI          | 综合性人工智能顶会。                                         |
| [KDD](https://kdd.org/)                   | 图 · 数据   | 偏应用和数据挖掘，图表征学习常见于此。                       |
